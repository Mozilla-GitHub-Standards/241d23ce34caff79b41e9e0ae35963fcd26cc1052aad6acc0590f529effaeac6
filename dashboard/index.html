Plugin Hangs in Firefox Desktop (Weekly)

This dashboard demonstrates what percentage of FHR records observe plugin hangs. We want to track Flash hangs, specifically, but FHR does not make a distinction between plugin hangs broadly and Flash hangs specifically. So baked into this dashboard is the assumption that a plugin hang tends to be a Flash hang.

These data are presented weekly.

Sadly, that segment contains insufficient data.

Users without Plugin Hangs

What percentage of records don't observe plugin hangs? How many plugin hangs occur per 200 FHR records? For those records that experience plugin hangs, how many on average do they observe each week?

To reiterate, these data are presented on a weekly time scale.

Average Session lengths Per Plugin Crash ("Mean Time Between Failures")

The chart on the right illustrates the mean time between failures, defined as the total session length divided by the number of plugin crashes. Shown here only reflects the records that have at least two plugin hangs.
